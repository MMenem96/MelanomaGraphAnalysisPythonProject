About the BCC vs SK Detection System

Research Background

This BCC vs SK detection system is based on the paper: "Melanoma Detection Using Spatial and Spectral Analysis on Superpixel Graphs" by Mahmoud H Annaby, Asmaa M Elwer, Muhammad A Rushdi, Mohamed E M Rasmy. The system uses advanced image processing techniques, graph theory, and machine learning to analyze dermoscopic images of skin lesions.

By representing skin lesions as graphs of superpixels and extracting both spatial and spectral features, our system can identify subtle patterns that help differentiate between Seborrheic Keratosis (SK) and Basal-cell Carcinoma (BCC).

Technical Approach

Before analysis, each image undergoes several preprocessing steps:

  • Color space conversion to LAB color space
  • Contrast enhancement to improve feature visibility
  • Noise reduction to remove artifacts
  • Size normalization to ensure consistent analysis

The system also validates that the uploaded image contains a skin lesion with appropriate characteristics.

The system divides the image into approximately 20 superpixels using the SLIC (Simple Linear Iterative Clustering) algorithm. Each superpixel represents a region with similar color and texture characteristics.

Superpixels provide a meaningful decomposition of the image that preserves important boundaries and structures within the lesion.

The superpixels are represented as nodes in a graph, with edges connecting similar regions. The edge weights are determined by the feature similarity between superpixels, including:

  • Color similarity in LAB space
  • Texture similarity using Haralick features
  • Spatial proximity between regions

This graph representation captures the structural relationships between different regions of the lesion.

The system extracts over 500 features from the graph and image, categorized as:

  • Graph Features (149 features): Local node measures (120), global graph measures (5), and spectral features (24)
  • Conventional Image Features (394 features): Color features (150), texture features (144), and geometric/shape features (100)

These features capture both the spatial and spectral characteristics of the lesion that help distinguish Basal-cell Carcinoma (BCC) from Seborrheic Keratosis (SK).

The extracted features are fed into a machine learning classifier (Support Vector Machine or Random Forest) that has been trained on a dataset of known Basal-cell Carcinoma (BCC) and Seborrheic Keratosis (SK) lesion images.

The classifier outputs a probability score indicating the likelihood that the lesion is Basal-cell Carcinoma (BCC).

Important evaluation metrics include accuracy, sensitivity, specificity, and the area under the ROC curve.

Important Disclaimer

This system is intended as a supporting tool for preliminary assessment and should not replace professional medical diagnosis. Always consult with a qualified dermatologist for proper evaluation of skin lesions.

The accuracy of the system depends on image quality, proper preprocessing, and adequate training data. Results should be interpreted with caution and in the appropriate clinical context.
